# React Native Agent Skill

**Skill Name:** `react-native-skills`  
**Registry Root:** `skills/react-native-skills/SKILL.md`

## Usage Guidelines
Apply this skill whenever building or optimizing React Native and Expo applications. It is designed to ensure mobile-first performance and architectural consistency.

## When to Use
- **Mobile Development:** Building new features or screens in RN/Expo.
- **Performance Tuning:** Optimizing lists, reducing re-renders, or fixing frame drops.
- **Animations:** Implementing high-performance UI transitions with Reanimated.
- **State & Architecture:** Structuring complex data flows and monorepo dependencies.
- **Native Modules:** Interfacing with platform-specific APIs or native components.

## Core Expertise Areas
1. **Performance:** Efficient `FlatList` usage, virtualization, and memoization.
2. **Animations:** Worklet-based logic and `useDerivedValue` patterns.
3. **Navigation:** Native-first navigation structures.
4. **UI/UX:** Modern interaction patterns using the `Pressable` API.
5. **State:** Minimizing local state and maintaining consistent data sources.
