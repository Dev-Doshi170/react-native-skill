# React Native & Expo Agent Skills

🚀 **Production-grade AI agent skills for high-performance React Native development.**

This repository contains a curated set of architectural rules and best practices optimized for AI coding agents. It ensures that any agent (Antigravity, Claude, Cursor, etc.) produces mobile code that is performant, maintainable, and reflects industry-standard patterns.

## 📦 Installation

To add these skills to your local project, run:

```bash
npx skills add https://github.com/Dev-Doshi170/react-native-skill --skill react-native-skills
```

## 🛠️ What it Covers

This skill registry provides deep expertise in:

- ⚡ **Performance Optimization**: Advanced FlatList virtualization, memoization strategies, and re-render minimization.
- 🎬 **Animations**: 60fps gesture-based animations using React Native Reanimated.
- 🧭 **Navigation**: Native Stack optimization and efficient navigation state handling.
- 🎨 **UI Patterns**: Implementation of the Pressable API, scalable StyleSheets, and consistent design tokens.
- 💾 **State Management**: Enforcing Single Source of Truth and efficient state distribution.
- 🏗️ **Architecture**: Monorepo-safe dependency management and clean code structure.
- 📱 **Platform-Specific Patterns**: Handling iOS and Android nuances gracefully.
