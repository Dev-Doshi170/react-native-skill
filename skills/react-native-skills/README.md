# Vercel-Style React Native Skills

This repository contains a curated library of React Native development rules, optimized for AI agents and senior developers.

## Structure

- `/rules`: Granular markdown files covering specific technical topics.
- `SKILL.md`: The active context definition.
- `metadata.json`: Skill configuration.

## Usage

These rules are intended to be loaded into your agent's context to enforce architectural standards during code generation.
