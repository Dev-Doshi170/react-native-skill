# Cursor Rules: React Native Skills

This repository uses a structured skill registry for React Native development.

## Project Rules
Always follow the guidelines in `skills/react-native-skills/SKILL.md` when:
- Creating new components or screens.
- Implementing list-based UIs (`FlatList`, `SectionList`).
- Handling complex animations or transitions.
- Managing application state and navigation.

## Key Focus Areas
- **Mobile Performance:** Memoization, virtualization, and avoiding inline objects.
- **Modern UI:** Consistent use of `Pressable` and `StyleSheet`.
- **Architecture:** Monorepo compatibility and dependency parity.

For specific code snippets and rule definitions, check the [SKILL.md](skills/react-native-skills/SKILL.md) file.
